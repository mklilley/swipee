### Front end

The Swipee front end is built using [Vue.js version 2](https://vuejs.org/v2/guide/). I used the [Vue CLI version 4.4.6](https://cli.vuejs.org/) to create the project:

- `npm install -g @vue/cli`
- `vue create swipee`
