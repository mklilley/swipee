### Front end

The Swipee front end is built using [Vue.js version 3](https://v3.vuejs.org/guide/introduction.html). I used the [Vue CLI version 4.5.11](https://cli.vuejs.org/) to create the project:

- `npm install -g @vue/cli`
- `vue create swipee`
